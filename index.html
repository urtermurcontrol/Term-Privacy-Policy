Privacy Policy

Data Collection and Storage

  • The extension does not collect or transmit any user data to external servers.
  • User privacy preferences are stored locally using Chrome's built-in storage APIs.
  • Button position and extension state are saved locally on the user's device.
  • If Chrome sync is enabled, preferences may sync across signed-in devices.

Website Content Analysis

  • All text analysis occurs solely on the user's device.
  • Content is analyzed only when the user explicitly clicks the TERM button.
  • No website content is transmitted to external servers.
  • Analysis results are not stored beyond the current browsing session.

User Control

  • Users have full control over when analysis occurs.
  • Privacy concern preferences can be customized.
  • The extension can be disabled or hidden at any time.
  • There is no automatic scanning of websites.

Data Sharing

  • No data is shared with third parties.
  • No analytics, telemetry, or behavior tracking is collected.

Technical Implementation

  • All code is contained within the extension package.
  • No remote code execution occurs.
  • No data is transmitted to external servers.

Permissions Used

  • The extension requires activeTab, scripting, and storage permissions for its core privacy analysis functionality.
  • These permissions are used exclusively for the analysis features described above.
